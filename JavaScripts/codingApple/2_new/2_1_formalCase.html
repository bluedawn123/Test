1. /abc/.test('abcdef') => /정규식/.test(정규식으로 검사해볼문자) 쓰면 됨. true or false

ex) /a/.test('abcde')

<2>2. [ ] 기호를 이용해서 문자 범위를 지정할 수 있습니다.

/[a-d]/.test('aefg') //true

/[가-다]/.test('다라마바') //true

3. [a-zA-Z] 이건 아무 알파벳 하나라는 뜻입니다.[ㄱ-ㅎ가-힣ㅏ-ㅣ] 이건 아무 한글 하나라는 뜻입니다.

/[ㄱ-ㅎ가-힣ㅏ-ㅣ]/.test('반가워요') //true

4.백슬래시S 는 특수문자 포함 아무문자 1개라는 뜻입니다. 여러개랑 쓰려면 + 가 필요

/\S/.test('abcde') //true

5.^ 라고 적으면 시작하는지 검사. => /^a/.test('abcde') //a로 시작하는지 검사하므로 참

6. $ 라고 적으면 e로 끝나는지 검사 => /e$/.test('abcde') //e로 끝나므로 참

7. | 이건 or 기호 => /(e|f)/.test('abcde') //true

8. + 기호를 붙여주시면 뒤에 오는 글자들도 a와 일치하면 반복해서 쭉 찾아달라는 뜻

/\S+t/ => 모든 문자 여러개 다음에 t라는 글자가 있냐를 검사해주는 정규식

/a+/ => aaaaa 이런걸 찾고 싶으면 /a+/

9 .을 사용하려면 \. 으로 써야한다.

답 => /\S+@\S+\.\S+/

해석 => 모든아무글자여러개 + @로 끝나면서 + 다시 모든문자여러개 + .으로 끝나면서 + 다시 아무문제여러개

예시 => asdfsadf @ naver . com

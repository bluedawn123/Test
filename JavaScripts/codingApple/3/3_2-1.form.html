셔츠 사이즈 선택






--------------------------------------------수업 내용-----------------------------------

수업내용 = form event들에 대해 배운다

기존 option들에서 선택하면 또다른 옵션이 나오게 구현 => 역시 특정상황에 작동하게 만들면 된다.

즉, 셔츠를 고르면 셔츠 사이즈를 선택할 수 있는 select 폼 구현

이벤트는 click, input, change 등등이 있다.

select는 선택만 해도 change이벤트가 발동한다,

input은 값이 바뀔때마다 발동된다.input은 입력후 커서를 떼면 발동된다.

사용자가 선택한 값이 셔츠인 경우 => val()를 사용해준다. 셀렉트면 $('select').val() 인풋이면 $('input').val()

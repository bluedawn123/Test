정규식 => / / => 문자를 검사할때 쓰는 정규식. if 와 같이 조건식처럼 써도 된다.

/abc/.test('qwer') => qwer 문자에 abc가 있나 검사해서 true or false(boolean형태로 나옴)

/[A-z]/.test('abcd') => 대문자A부터 소문자z까지 검증한 것에 abcd가 있는지 검증

/\S/ => 특수문자 포함 모든 문자. 그냥 \S 를 모든문자하나 라고 보자.

/\S@/.test('') => 특수문자 포함 모든 문자 다음에 골뱅이

/\S+@\S+\.\S+/.test('haemilyjh@naver.com') => 모든문자(\S+)+골뱅이+모든문자(\S+)+.(\.)+모든문자(\S+) 의미

이메일 정규식 => /^(([^<>()\[\]\.,;:\s@\"]+(\.[^<>()\[\]\.,;:\s@\"]+)*)|(\".+\"))@(([^<>()\.,;\s@\"]+\.{0,1})+([^<>()\.,;:\s@\"]{2,}|[\d\.]+))$/

es6로는 class를 사용해서 상속할 수 있다.

constructor 안에서, 밖에서 함수를 사용할 수 있다.

차이점은, 안에서 사용 경우 : 물려받는다. 자식이 직접 함수를 갖는다 => 부모 {name: '아이배고파', sayHi1: ƒ}

밖에서 사용 경우 : 물려받지 않는다. 부모의 prototype에 추가된다.. => 모든 자식들이 공통적으로 사용가능

참고로 자식.__proto__ = Object.getPrototypeOf(자식) = 부모.prototype 다 같다.

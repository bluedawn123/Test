수업내용 : 정렬기능을 위한 sort, map, filter함수에 대해 알아보자.

방법 => products array를 정렬하고 products에 있던걸로 html 재생성

1. sort = 여러 정렬 함수

2. filter = 원하는 자료만 필터하는 함수

3. map = array 안의 자료들을 전부 변형하려면 map 함수

-------------------------------------------------------------

sort의 오름차순 원리에 대해 알아보자

var 어레이1 = [7,3,5,2,40]

어레이1.sort(function(a,b){

return a - b //내림차순은 b - a

});

console.log(어레이1) 를 하면 2,3,5,7,40 이렇게 오름차순이 잘 나온다. 왜 그럴까?

return 오른쪽이 양수면 a를 오른쪽으로 정렬, return 오른쪽이 음수면 b를 오른쪽으로 정렬

리고 array 안의 자료들을 계속 뽑아서 a, b에 넣어줍니다.

이렇게 동작해서 a - b 저렇게 쓰면 숫자순 정렬

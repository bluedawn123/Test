입력양식은 버튼, 글자입력, 선택, 체크박스, 라디오 등이 있다.

버튼으로 사용할 수 있는 것은 크게 3가지. 1. 버튼태그 2.인풋 타입중 버튼이라는 타입의 태그 3. 인풋태그 중 서밋이라는 태그

1, 2번째 방식은 클릭이벤트를 활용하고 3번째는 submit 이벤트를 활용한다.

input type submit??

form이라는 것은 입력양식이다. form을 만들때 form태그를 사용하고 제출시 submit 버튼을 사용한다. form태그 입력양식에 입력하고 submit버튼을 누르면 주소가 바뀐다. 즉 데이터를 다른 페이지(서버)로 넘긴다.

submit이벤트는 input 타입 submit에 연결하는 것이 아니라 document.queryselector로 form태그를 잡고 form태그에 연결을 해준다.

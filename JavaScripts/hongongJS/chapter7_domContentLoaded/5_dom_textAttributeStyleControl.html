1. 글자 조작(추출 or 조작) 하는 방법 => 크게 2가지. 1.textContent 2. innerHTML 두 속성은 내부에 html의 유무에 따라 달라진다. textContent는 그대로 나오고 innerHTML은 적용돼서 나타난다.

2. 속성 조작 => 역시 값을 넣거나 조작한다. 값을 넣을때는 setAttribute('속성이름', '값').

==========>표준인 경우는 img.src등과 같이 간단하게 바로 사용할 수 있다. 표준에 없는 속성을 조작할때는 사용자정의속성(데이터속성) ex)img src="" alt="" 정보=""에서 정보를 활용하게 된다. 일단 표준에 없으면 get, set을 활용한다고 알아두자.

3. 스타일 조작

4. 정리 : 글자조작 => textContent, innerHTML. /// 속성 조작 : setAttribute('이름', '값'), getAttribute('이름'), 표준에있으면 그냥 입력 ///////// 스타일 조작 : style.backgroundColor, sytle.기타속성입력
